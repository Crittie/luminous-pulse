## On-Page SEO Checklist (Every Post)

- [ ] Primary keyword in H1 title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in meta description (150-160 chars)
- [ ] Primary keyword in URL slug
- [ ] Secondary keywords in H2/H3 headers naturally
- [ ] Alt text with keyword on images
- [ ] 3-5 internal links to related posts
- [ ] 3-5 external links to authoritative sources
- [ ] FAQ section with FAQPage schema
- [ ] Author bio with credentials
- [ ] CTA to lead magnet (in-content + end-of-post)
- [ ] CTA to paid product (end-of-post)
- [ ] Table of contents for posts 2,500+ words
- [ ] Short paragraphs (2-4 sentences)
- [ ] Subheaders every 200-300 words
- [ ] At least one custom image

## Schema Markup Templates

### Article Schema (every post)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "[Post Title]",
  "author": {
    "@type": "Person",
    "name": "Christie Parrow",
    "jobTitle": "PMP-Certified Project Manager, Founder of Luminous Pulse"
  },
  "datePublished": "[Date]",
  "publisher": {
    "@type": "Organization",
    "name": "Luminous Pulse",
    "url": "https://luminouspulse.co"
  }
}
```

### FAQPage Schema (posts with FAQ sections)
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "[Question text]",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "[Answer text]"
    }
  }]
}
```

### HowTo Schema (tutorial posts)
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "[Tutorial title]",
  "step": [{
    "@type": "HowToStep",
    "name": "[Step name]",
    "text": "[Step description]"
  }]
}
```

## Content-to-Product Funnel

Every blog post includes:

1. **In-content CTA** (after introduction): Soft link to free lead magnet
2. **Mid-content resource box**: Related to post topic
3. **End-of-post conversion**: Lead magnet form + paid product link
4. **Email nurture**: Blog subscriber → 5-Day Practice → Welcome sequence → Nurture → Paid product
5. **Course enrollment CTA**: On posts about structured recovery and long-term practice, link to "grounded through change" course ($197) or waitlist
